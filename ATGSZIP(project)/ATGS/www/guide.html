Automated Timetable Generation System

 

 

 

 

 

             1. Launch the Easyphp Application.

           2. When Easyphp app is successfully started and running smoothly.

           3. Open the ATGS software on browser using localhost url.

           4. Now login with Key given with the system .

           5. If username or password is wrong then you will be redirected to forget

               password page.

           6. If  password and username are correct then you will be redirected to

               Admin panel.

           7. Now you can add faculty, subjects ,classrooms by visiting there panels .

           8. After the input of data you must have to visit allocation panel and have to

               allocate theory & practcals subject to there faculties, after this you have to

               allocate classes to the classroom where they ware taken.

           9. After allocation you must visit the admin panel and then visit the modify

               timetable panel.

           10. Now you can reset and update the faculty timetable and also you can

                 update the semester table.

                      (Note - You must have to reset all the teacher table before generation of

                                 Timetable of other session.)

           11. After that visit the admin panel and then go to generation page, now first

                 Set the session and then press the “Generate” button it must take more then

                 8 minutes to generate the timetable of 3 classes at a time.

           12. After that you can view timetable by teachers and by semesters.

           13. You can print/save timetable by pressing save as pdf button.

           14. After all that you can register new admin user by visiting the admin

                 registration page also you can update your account info by visiting the

                 manage account page.

           15. Now you can logout and go to homepage using logout button.
